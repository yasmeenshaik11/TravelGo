TravelGo – Cloud Powered Travel Booking Platform
Overview

TravelGo is a full-stack travel booking web application that allows users to search and book buses, trains, flights, and hotels through a single platform.
The system provides a simple interface for users to register, log in, browse travel options, and manage their bookings.

This project is built using Flask (Python) for the backend and HTML, CSS, and JavaScript for the frontend.

Features
User Features

User registration and login

Search travel options

Book transportation and hotels

View booking history

Personal dashboard

Responsive UI

Admin / Owner Features

Add travel packages

Remove travel options
